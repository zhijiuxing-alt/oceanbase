parallel_servers_target 
============================================

parallel_servers_target 用于设置每个 Server 上的大查询排队条件。当并行执行（Parallel eXecution，PX）线程池中有指定的空闲线程数时才调度新查询。


| **属性** |           **描述**           |
|--------|----------------------------|
| 参数类型   | int                        |
| 默认值    | 0                          |
| 取值范围   | \[0, 9223372036854775807\] |
| 生效范围   | GLOBAL                     |

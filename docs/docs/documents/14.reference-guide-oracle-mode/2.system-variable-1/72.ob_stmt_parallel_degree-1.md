ob_stmt_parallel_degree 
============================================

ob_stmt_parallel_degree 用于设置查询的并行度，即可以并行运行的任务数。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | int                                                                                                        |
| 默认值      | 1                                                                                                          |
| 取值范围     | \[1, 10240\]                                                                                               |
| 生效范围     | * GLOBAL   * SESSION    |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |

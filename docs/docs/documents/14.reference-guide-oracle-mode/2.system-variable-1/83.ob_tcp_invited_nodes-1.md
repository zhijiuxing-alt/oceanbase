ob_tcp_invited_nodes 
=========================================

`ob_tcp_invited_nodes` 用于设置租户的 IP 白名单，支持百分号（%） 、短横线（_） 和 IP（多个 IP 之间用逗号分隔），支持 IP 列表匹配、掩码匹配和模糊匹配。


| **属性** |    **描述**     |
|--------|---------------|
| 参数类型   | varchar       |
| 默认值    | 127.0.0.1,::1 |
| 取值范围   | 字符串长度小于 1024  |
| 生效范围   | GLOBAL        |

debug_sync 
===============================

debug_sync 用于在 debug 时设置同步点，与 MySQL 兼容。
**说明**



只有 debug_sync_timeout 配置项的值不为 0 时，该变量的设置才有效。


| **属性** |   **描述**   |
|--------|------------|
| 参数类型   | varchar    |
| 默认值    | 空字符串       |
| 取值范围   | 无          |
| 生效范围   | SESSION    |
| 是否可见   | 否，所有租户均不可见 |

nls_characterset 
=====================================

nls_characterset 表示数据库默认字符集，用于 CHAR、VARCHAR2、LONG、CLOB 等数据类型。


|  **属性**  |  **描述**  |
|----------|----------|
| 参数类型     | varchar  |
| 默认值      | AL32UTF8 |
| 取值范围     | AL32UTF8 |
| 生效范围     | GLOBAL   |
| 是否参与序列化  | 是        |
| 是否影响计划生成 | 是        |
| 是否可修改    | 否        |

ob_safe_weak_read_snapshot 
===============================================

ob_safe_weak_read_snapshot 用于 Proxy 向数据库同步弱读一致性读快照。


| **属性**  |           **描述**           |
|---------|----------------------------|
| 参数类型    | int                        |
| 默认值     | 1                          |
| 取值范围    | \[0, 9223372036854775807\] |
| 生效范围    | SESSION                    |
| 是否参与序列化 | 是                          |
| 是否可见    | 否，对所有租户均不可见                |

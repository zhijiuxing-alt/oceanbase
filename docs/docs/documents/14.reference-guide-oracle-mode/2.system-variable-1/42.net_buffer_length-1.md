net_buffer_length 
======================================

net_buffer_length 用于设置 OBServer 能接受的 SQL Query 的大小。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                                                                                                        |
| 默认值    | 16384                                                                                                      |
| 取值范围   | \[1024, 1048576\]                                                                                          |
| 生效范围   | * GLOBAL   * SESSION    |


**说明**



该变量不能通过 `alter session set xxx = yyy; `语句的方式使其仅在当前 Session 生效，只能通过 `set global xxx = yyy` 语句的方式使全局生效。

ob_read_consistency 
========================================

ob_read_consistency 用于设置读一致性级别。


|  **属性**  |                                                                                                **描述**                                                                                                 |
|----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型     | enum                                                                                                                                                                                                  |
| 默认值      | 3                                                                                                                                                                                                     |
| 取值范围     | * 0：空字符串   * 1：FROZEN   * 2：WEAK   * 3：STRONG    |
| 生效范围     | * GLOBAL   * SESSION                                                                                               |
| 是否参与序列化  | 是                                                                                                                                                                                                     |
| 是否影响计划生成 | 是                                                                                                                                                                                                     |

ob_capability_flag 
=======================================

ob_capability_flag 是 OBServer 能够提供的能力，用于和 Proxy 连接时进行能力协商，告知 Proxy 当前 OBServer 支持哪些功能，不支持哪些功能。


| **属性** |           **描述**           |
|--------|----------------------------|
| 参数类型   | uint                       |
| 默认值    | 0                          |
| 取值范围   | \[0,18446744073709551615\] |
| 生效范围   | SESSION                    |
| 是否可修改  | 否                          |
| 是否可见   | 否，对所有租户均不可见                |

plugin_dir 
===============================

plugin_dir 用于设置存放插件 DLL 的路径。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | varchar                                                                                                    |
| 默认值    | ./plugin_dir/                                                                                              |
| 取值范围   | NULL                                                                                                       |
| 生效范围   | * GLOBAL   * SESSION    |

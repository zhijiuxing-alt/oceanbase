ob_create_table_strict_mode 
================================================

`ob_create_table_strict_mode` 是动态变量，用于设置建表的严格模式。


| **属性** |                                                                                   **描述**                                                                                    |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                                                                                        |
| 默认值    | 1                                                                                                                                                                           |
| 取值范围   | * 1：表示会严格按照 Locality 建立副本，任何副本建立失败，则建表失败。   * 0：表示全类型副本至少有 1 个，Paxos 成员组副本达到多数，则建表成功。    |
| 生效范围   | * GLOBAL   * SESSION                                                                     |

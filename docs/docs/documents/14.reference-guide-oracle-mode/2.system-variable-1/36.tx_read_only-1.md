tx_read_only 
=================================

tx_read_only 用于设置是否为只读事务。


| **属性**  |                                                   **描述**                                                    |
|---------|-------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                         |
| 默认值     | 0                                                                                                           |
| 取值范围    | * 1：只读事务   * 0：不是只读事务    |
| 生效范围    | * GLOBAL   * SESSION     |
| 是否参与序列化 | 是                                                                                                           |


**说明**



设置后，仅影响该用户，不影响其他用户。

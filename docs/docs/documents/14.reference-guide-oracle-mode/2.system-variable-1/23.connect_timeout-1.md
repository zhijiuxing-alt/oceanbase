connect_timeout 
====================================

connect_timeout 用于设置连接超时时间，单位为微妙。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                                                                                                        |
| 默认值    | 10                                                                                                         |
| 取值范围   | \[2, 31536000\]                                                                                            |
| 生效范围   | * GLOBAL   * SESSION    |

nls_timestamp_format 
=========================================

nls_timestamp_format 用于控制 timestamp 或 timestamp ltz 类型转 str 的格式，以及 str 隐式转 timestamp 或 timestamp ltz 的格式。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | DD-MON-RR HH.MI.SSXFF AM                                                                                   |
| 取值范围    | NULL                                                                                                       |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |

nls_date_format 
====================================

nls_date_format 用于控制 date 类型转 str 的格式，以及 str 隐式转 date 的格式。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | DD-MON-RR                                                                                                  |
| 取值范围    | NULL                                                                                                       |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |

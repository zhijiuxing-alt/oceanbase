nls_dual_currency 
======================================

`nls_dual_currency` 用于设置数字格式元素 `U` 的本地货币符号。


|   属性    |                                                     描述                                                     |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | $                                                                                                          |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | * Global   * Session    |
| 是否参与序列化 | 是，即远程执行、分布式执行场景下，变量会参与序列化。                                                                                 |
| 是否重启生效  | 否                                                                                                          |

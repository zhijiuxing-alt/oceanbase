nls_nchar_conv_excp 
========================================

nls_nchar_conv_excp 用于控制 NCHAR/NVARCHAR2 与 CHAR/VARCHAR2 之间转换丢失数据时是否报错。


|  **属性**  |                                                      **描述**                                                       |
|----------|-------------------------------------------------------------------------------------------------------------------|
| 参数类型     | bool                                                                                                              |
| 默认值      | FALSE                                                                                                             |
| 取值范围     | * TRUE：表示报错   * FALSE：表示不报错    |
| 生效范围     | * GLOBAL   * SESSION           |
| 是否参与序列化  | 是                                                                                                                 |
| 是否影响计划生成 | 是                                                                                                                 |

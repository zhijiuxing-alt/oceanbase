character_set_system 
=========================================

character_set_system 设置服务器使用的字符集。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                                                                                                     |
| 默认值    | utf8mb4                                                                                                    |
| 取值范围   | utf8mb4                                                                                                    |
| 生效范围   | * GLOBAL   * SESSION    |
| 是否可以修改 | 否                                                                                                          |

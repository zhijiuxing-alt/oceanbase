nls_comp 
=============================

nls_comp 表示字符串值的比较规则。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | varchar                                                                                                    |
| 默认值      | BINARY                                                                                                     |
| 取值范围     | BINARY                                                                                                     |
| 生效范围     | * GLOBAL   * SESSION    |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |

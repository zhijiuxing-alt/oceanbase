ob_enable_truncate_flashback 
=================================================

ob_enable_truncate_flashback 用于设置是否启用表截断的闪回。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                       |
| 默认值    | 0                                                                                                          |
| 取值范围   | * 0：不启用   * 1：启用        |
| 生效范围   | * GLOBAL   * SESSION    |

ob_trx_timeout 
===================================

ob_trx_timeout 用于设置事务超时时间，单位为微秒。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | 100000000                                                                                                  |
| 取值范围    | NULL                                                                                                       |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |

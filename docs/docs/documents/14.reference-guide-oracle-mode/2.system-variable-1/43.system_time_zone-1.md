system_time_zone 
=====================================

system_time_zone 用于设置服务器机器系统时区。


| **属性**  |                  **描述**                   |
|---------|-------------------------------------------|
| 参数类型    | varchar                                   |
| 默认值     | CST                                       |
| 取值范围    | N/A                                       |
| 生效范围    | GLOBAL                                    |
| 是否可修改   | 否，仅在 OBServer 启动的命令行上指定，启动后不能再通过 SQL 语句修改 |
| 是否参与序列化 | 是                                         |

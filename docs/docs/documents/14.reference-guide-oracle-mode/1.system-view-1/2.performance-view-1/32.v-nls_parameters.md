V$NLS_PARAMETERS 
=====================================



**功能** 
---------------------------

查看 Session 级别的 NLS 参数。

**相关视图** 
-----------------------------

* NLS_INSTANCE_PARAMETERS

  

* NLS_DATABASE_PARAMETERS

  

* NLS_SESSION_PARAMETERS

  




**字段说明** 
-----------------------------



| **字段名称**  |     **类型**      | **是否可以为 NULL** | **描述** |
|-----------|-----------------|----------------|--------|
| PARAMETER | VARCHAR2(128)   | NO             | 参数名称   |
| VALUE     | VARCHAR2(65536) | YES            | 参数取值   |
| CON_ID    | NUMBER          | NO             | 无实际含义  |

---------------------------

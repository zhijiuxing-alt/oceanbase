V$NLS_PARAMETERS 
=====================================



**功能** 
---------------------------

查看租户级别的 NLS 参数。

**字段说明** 
-----------------------------



| **字段名称**  |     **类型**     | **是否可以为 NULL** |     **描述**      |
|-----------|----------------|----------------|-----------------|
| PARAMETER | varchar(128)   | NO             | 参数名称            |
| VALUE     | varchar(65536) | YES            | 参数取值            |
| CON_ID    | bigint(1)      | NO             | 容器 ID。目前该字段暂未使用 |

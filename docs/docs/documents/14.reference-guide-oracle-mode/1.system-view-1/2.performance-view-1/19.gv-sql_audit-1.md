GV$SQL_AUDIT 
=================================



**功能** 
---------------------------

该视图用于展示所有 Server 上每一次 SQL 请求的来源、执行状态等统计信息。该视图是按照租户拆分的，除了系统租户，其他租户不能跨租户查询。

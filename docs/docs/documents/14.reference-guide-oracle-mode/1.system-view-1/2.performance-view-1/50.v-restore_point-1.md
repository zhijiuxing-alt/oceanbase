V$RESTORE_POINT 
====================================



**功能** 
---------------------------

显示有关还原点的信息。

**字段说明** 
-----------------------------



|   字段名称    |                类型                 | **是否可以为 NULL** |     **描述**     |
|-----------|-----------------------------------|----------------|----------------|
| TENANT_ID | NUMBER(38)                        | YES            | 租户 ID          |
| SCN       | NUMBER(38)                        | NO             | 创建还原点时的数据库 SCN |
| TIME      | TIMESTAMP(6) WITH LOCAL TIME ZONE | NO             | 创建还原点时的时间      |
| NAME      | VARCHAR2(512)                     | YES            | 还原点名称          |

ALL_DEF_AUDIT_OPTS 
=======================================



功能 
-----------

记录任何对象默认被审计的审计条件。每列的取值如下：

* `-/-`：默认不审计

* `S/-`: 只在语句执行成功时审计

* `-/S`: 只在语句执行失败时审计

* `S/S`: 语句执行无论成功还是失败时都审计




相关视图 
-------------

DBA_OBJ_AUDIT_OPTS

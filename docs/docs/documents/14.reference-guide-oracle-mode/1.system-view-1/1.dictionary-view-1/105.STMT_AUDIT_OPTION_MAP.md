STMT_AUDIT_OPTION_MAP 
==========================================



功能 
-----------

记录语句级别审计的 Type 及对应的名称和优先级。

相关视图 
-------------

DBA_STMT_AUDIT_OPTS

字段说明 
-------------



| **字段名称** |    **类型**     | **是否可以为 NULL** |               **描述**                |
|----------|---------------|----------------|-------------------------------------|
| OPTION#  | NUMBER(38)    | NO             | 语句级别审计的 Type ID                     |
| NAME     | VARCHAR2(128) | NO             | 语句级别审计的 Type 对应的名称                  |
| PROPERTY | NUMBER        | NO             | 语句级别审计 Type 的优先级，目前暂不支持该字段，默认为 NULL |

AUDIT_ACTIONS 
==================================



功能 
-----------

记录审计记录中 action id 和对应的名称。

相关视图 
-------------

无

字段说明 
-------------



| **字段名称** |    **类型**     | **是否可以为 NULL** |          **描述**          |
|----------|---------------|----------------|--------------------------|
| ACTION   | NUMBER(38)    | NO             | 审计记录中的 Action ID         |
| NAME     | VARCHAR2(128) | NO             | 审计记录中 Action ID 对应的 Name |

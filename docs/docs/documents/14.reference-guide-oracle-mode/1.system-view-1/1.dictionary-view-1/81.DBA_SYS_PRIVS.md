DBA_SYS_PRIVS 
==================================



功能 
-----------

描述授予用户和角色的系统特权。此视图不显示 `USERNAME` 列。

相关视图 
-------------

USER_SYS_PRIVS

字段说明 
-------------



|   **字段名称**   |    **类型**     | **是否可以为 NULL** |      **描述**      |
|--------------|---------------|----------------|------------------|
| GRANTEE      | VARCHAR2(128) | NO             | 接受授权的授予者名称、用户或角色 |
| PRIVILEGE    | VARCHAR2(40)  | NO             | 系统权限             |
| ADMIN_OPTION | VARCHAR2(3)   | NO             | 授予管理选项           |

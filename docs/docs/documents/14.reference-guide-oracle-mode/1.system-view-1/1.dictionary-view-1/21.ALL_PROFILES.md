ALL_PROFILES 
=================================



功能 
-----------

查看所有的简介和这些简介的限制。

相关视图 
-------------

* DBA_PROFILES

  

* USER_PROFILES

  




字段说明 
-------------



|   **字段名称**    |    **类型**    | **是否可以为 NULL** |              **描述**               |
|---------------|--------------|----------------|-----------------------------------|
| PROFILE       | VARCHAR2(30) | NO             | 简介的名称                             |
| RESOURCE_NAME | VARCHAR2(32) | NO             | 资源的名称                             |
| RESOURCE_TYPE | VARCHAR2(8)  | NO             | 表明一个资源的简介是一个 KERNEL 还是一个 PASSWORD |
| LIMIT         | VARCHAR2(40) | NO             | 简介资源的限制                           |

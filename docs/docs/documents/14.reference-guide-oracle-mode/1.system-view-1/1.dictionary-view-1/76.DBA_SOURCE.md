DBA_SOURCE 
===============================



功能 
-----------

列出数据库中可用的过程和函数的源代码。

相关视图 
-------------

* ALL_SOURCE

  

* USER_SOURCE

  




字段说明 
-------------



|   **字段名称**    |    **类型**    | **是否可以为 NULL** |       **描述**       |
|---------------|--------------|----------------|--------------------|
| OWNER         | VARCHAR2(30) | NO             | 对象所属的拥有者           |
| NAME          | VARCHAR2(30) | NO             | 对象名                |
| TYPE          | VARCHAR2(12) | NO             | 对象类型               |
| LINE          | NUMBER       | NO             | TEXT 字段对应的源代码中的行号  |
| TEXT          | CLOB         | NO             | 当前行的源代码            |
| ORIGIN_CON_ID | NUMBER(38)   | NO             | 暂不支持该字段，字段默认为 NULL |

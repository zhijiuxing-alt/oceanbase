USER_COL_COMMENTS 
======================================



功能 
-----------

查看当前用户拥有的视图和表的所有列注释。

相关视图 
-------------

* ALL_COL_COMMENTS

  

* DBA_COL_COMMENTS

  




字段说明 
-------------



|  **字段名称**   |     **类型**     | **是否可以为 NULL** | **描述** |
|-------------|----------------|----------------|--------|
| OBJECT_NAME | VARCHAR2(128)  | NO             | 对象名字   |
| COLUMN_NAME | VARCHAR2(128)  | NO             | 列名     |
| COMMENTS    | VARCHAR2(4000) | YES            | 列注释    |

ROLE_SYS_PRIVS 
===================================



功能 
-----------

描述授予角色的系统特权。

字段说明 
-------------



|   **字段名称**   |    **类型**     | **是否可以为 NULL** |       **描述**       |
|--------------|---------------|----------------|--------------------|
| ROLE         | VARCHAR2(128) | NO             | 角色名称               |
| PRIVILEGE    | VARCHAR2(40)  | NO             | 授予该角色的系统权限         |
| ADMIN_OPTION | VARCHAR2(3)   | NO             | 该角色是否被授予了 ADMIN 选项 |

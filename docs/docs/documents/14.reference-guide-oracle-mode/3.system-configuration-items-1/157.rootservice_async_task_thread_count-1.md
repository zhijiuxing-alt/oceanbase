rootservice_async_task_thread_count 
========================================================

rootservice_async_task_thread_count 用于设置 RootService 内部异步任务使用的线程池大小。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 4         |
| 取值范围             | \[1, 10\] |
| 是否重启 OBServer 生效 | 否         |

default_row_format 
=======================================

default_row_format 用于设置在 MySQL 模式下，建表时使用的默认行格式。


|      **属性**      |                                                                                                  **描述**                                                                                                   |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                                                                                       |
| 默认值              | DYNAMIC                                                                                                                                                                                                   |
| 取值范围             | * REDUNDANT   * COMPAT   * DYNAMIC   * COMPRESSED    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                         |

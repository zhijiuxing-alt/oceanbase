syslog_io_bandwidth_limit 
==============================================

syslog_io_bandwidth_limit 用于设置系统日志所能占用的磁盘 IO 带宽上限，超过带宽上限容量的系统日志将被丢弃。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 容量单位   |
| 默认值              | 30MB   |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |

max_string_print_length 
============================================

max_string_print_length 用于设置打印系统日志时，单行日志最大长度。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 500      |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 生效 | 否        |

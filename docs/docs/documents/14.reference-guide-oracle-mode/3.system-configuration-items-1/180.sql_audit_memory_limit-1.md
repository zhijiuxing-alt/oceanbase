sql_audit_memory_limit 
===========================================

sql_audit_memory_limit 用于设置 SQL 审计数据可占用的最大内存。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 容量单位       |
| 默认值              | 3G         |
| 取值范围             | \[64M, +∞) |
| 是否重启 OBServer 生效 | 否          |

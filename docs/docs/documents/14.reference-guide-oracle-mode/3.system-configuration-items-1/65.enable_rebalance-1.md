enable_rebalance 
=====================================

enable_rebalance 用于设置是否开启自动负载均衡的功能。


|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                          |
| 默认值              | True                                                                                                          |
| 取值范围             | * True：开启   * False：不开启    |
| 是否重启 OBServer 生效 | 否                                                                                                             |

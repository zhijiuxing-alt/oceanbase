plsql_ccflags 
==================================

plsql_ccflags 用于定义 PL 编译器的编译常量。


|        属性        |       描述        |
|------------------|-----------------|
| 参数类型             | 字符串类型           |
| 默认值              | 空字符串            |
| 取值范围             | 符合 PL 编译器规范的字符串 |
| 是否重启 OBServer 生效 | 否               |


**说明**



该配置项可以任意定义变量，建议用于控制条件编译。

使用示例如下：

```sql
obclient> ALTER SESSION SET PLSQL_CCFLAGS = 'DeBug:TRUE';

obclient> ALTER SYSTEM SET PLSQL_CCFLAGS = 'DeBug:TRUE';
```

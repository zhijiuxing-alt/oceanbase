large_query_threshold 
==========================================

large_query_threshold 用于设置查询执行时间的阈值。超过时间的请求可能被暂停，暂停后自动被判断为大查询，执行大查询调度策略。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 5s         |
| 取值范围             | \[1ms, +∞) |
| 是否重启 OBServer 生效 | 否          |

cluster_id 
===============================

cluster_id 用于设置本 OceanBase 集群 ID。


|      **属性**      |      **描述**       |
|------------------|-------------------|
| 参数类型             | 整型                |
| 默认值              | 0                 |
| 取值范围             | \[1, 4294901759\] |
| 是否重启 OBServer 生效 | 否                 |

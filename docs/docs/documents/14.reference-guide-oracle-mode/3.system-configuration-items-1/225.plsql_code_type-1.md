plsql_code_type 
====================================

plsql_code_type 用于指定 PL/SQL 代码的编译类型。


|        属性        |                                                            描述                                                            |
|------------------|--------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                    |
| 默认值              | NATIVE                                                                                                                   |
| 取值范围             | * INTERPETED：解析型   * NATIVE：二进制可执行    |
| 是否重启 OBServer 生效 | 否                                                                                                                        |



使用示例如下：

```sql
obclient> ALTER SESSION SET plsql_code_type='INTERPRETED';
```

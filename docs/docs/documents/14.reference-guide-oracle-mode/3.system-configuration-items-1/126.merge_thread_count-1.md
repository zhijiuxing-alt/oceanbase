merge_thread_count 
=======================================

`merge_thread_count` 用于设置每日合并工作的线程数。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 0          |
| 取值范围             | \[0, 256\] |
| 是否重启 OBServer 生效 | 否          |



该配置项的值为 `0` 时，合并的工作进程数的计算方式为 `min{10,cpu_cnt*0.3}`，其中 `cpu_cnt` 为系统 CPU 的数量。

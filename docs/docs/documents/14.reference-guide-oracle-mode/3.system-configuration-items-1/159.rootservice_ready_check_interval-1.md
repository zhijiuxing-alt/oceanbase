rootservice_ready_check_interval 
=====================================================

rootservice_ready_check_interval 用于设置 RootService 启动后等待和检查集群状态的时间间隔。


|      **属性**      |      **描述**      |
|------------------|------------------|
| 参数类型             | 时间类型             |
| 默认值              | 3s               |
| 取值范围             | \[100000us, 1m\] |
| 是否重启 OBServer 生效 | 否                |

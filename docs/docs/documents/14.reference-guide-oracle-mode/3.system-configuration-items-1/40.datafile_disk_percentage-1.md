datafile_disk_percentage 
=============================================

datafile_disk_percentage 表示占用 data_dir 所在磁盘（磁盘 data_dir 所在磁盘将被OceanBase 数据库系统初始化用于存储数据）总空间的百分比。


| **属性** |  **描述**  |
|--------|----------|
| 参数类型   | 整型       |
| 默认值    | 90       |
| 取值范围   | \[5，99\] |
| 是否可修改  | 否        |

backup_backup_archivelog_retry_interval 
============================================================

backup_backup_archivelog_retry_interval 用于设置备份备份日志的任务的重试间隔。


|        属性        |    描述     |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 0s        |
| 取值范围             | \[0s,1h\] |
| 是否重启 OBServer 生效 | 否         |

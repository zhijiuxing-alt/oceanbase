plsql_v2_compatibility 
===========================================

plsql_v2_compatibility 用于指定 Oracle 8 的兼容功能。


|        属性        |                                                      描述                                                       |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                          |
| 默认值              | False                                                                                                         |
| 取值范围             | * True：兼容   * False：不兼容    |
| 是否重启 OBServer 生效 | 否                                                                                                             |



使用示例如下：

```sql
obclient> ALTER SESSION SET plsql_v2_compatibility=false;
```

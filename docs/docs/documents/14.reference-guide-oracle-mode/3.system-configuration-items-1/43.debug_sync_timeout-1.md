debug_sync_timeout 
=======================================

debug_sync_timeout 用于打开 debug sync 调试开关，并设置其超时时间。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 时间类型     |
| 默认值              | 0        |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 生效 | 否        |


**说明**



debug_sync_timeout 的值为 0 时，表示关闭 debug sync 调试。

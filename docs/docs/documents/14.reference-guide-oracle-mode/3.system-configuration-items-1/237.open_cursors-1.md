open_cursors 
=================================

open_cursors 用于限制单个 Session 可以同时打开的游标数量。


|        属性        |     描述      |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 50          |
| 取值范围             | \[0,65535\] |
| 是否重启 OBServer 生效 | 否           |

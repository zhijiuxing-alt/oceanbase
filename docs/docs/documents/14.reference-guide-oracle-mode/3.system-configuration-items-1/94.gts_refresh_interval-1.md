gts_refresh_interval 
=========================================

gts_refresh_interval 用于获取刷新全局时间戳服务的时间间隔。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 100us        |
| 取值范围             | \[10us, 1s\] |
| 是否重启 OBServer 生效 | 否            |

force_refresh_location_cache_interval 
==========================================================

force_refresh_location_cache_interval 用于设置刷新位置缓存的最大时间间隔。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 2h，表示 2 小时 |
| 取值范围             | \[1s, +∞)  |
| 是否重启 OBServer 生效 | 否          |

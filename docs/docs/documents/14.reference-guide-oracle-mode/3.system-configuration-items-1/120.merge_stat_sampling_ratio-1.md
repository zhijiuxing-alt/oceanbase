merge_stat_sampling_ratio 
==============================================

merge_stat_sampling_ratio 用于设置合并时候数据列统计信息的采样率。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 100        |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 生效 | 否          |

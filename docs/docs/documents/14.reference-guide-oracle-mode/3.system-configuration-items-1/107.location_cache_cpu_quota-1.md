location_cache_cpu_quota 
=============================================

location_cache_cpu_quota 用于设置位置缓存模块使用的 CPU 配额。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值              | 5         |
| 取值范围             | \[0, 10\] |
| 是否重启 OBServer 生效 | 否         |

net_thread_count 
=====================================

net_thread_count 用于设置网络 I/O 线程数。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 12         |
| 取值范围             | \[1, 100\] |
| 是否重启 OBServer 生效 | 否          |



当 net_thread_count 配置项值为 8 时，表示 RPC 和 MySQL 的 libeasy 的 I/O 线程各为 8 个线程。

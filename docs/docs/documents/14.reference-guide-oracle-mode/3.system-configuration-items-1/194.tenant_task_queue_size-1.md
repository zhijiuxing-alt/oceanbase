tenant_task_queue_size 
===========================================

tenant_task_queue_size 用于设置每个租户的请求队列大小。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 65536       |
| 取值范围             | \[1024, +∞) |
| 是否重启 OBServer 生效 | 否           |

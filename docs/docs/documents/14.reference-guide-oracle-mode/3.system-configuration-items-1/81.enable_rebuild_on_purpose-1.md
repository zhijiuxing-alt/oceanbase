enable_rebuild_on_purpose 
==============================================

enable_rebuild_on_purpose 在备副本日志落后过多时，设置是否开启主动触发副本重建操作。


|      **属性**      |                                              **描** **述**                                               |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | * True   * False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

trx_force_kill_threshold 
=============================================

trx_force_kill_threshold 用于设置因冻结或切主需要杀事务时的最长等待时间。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 100ms        |
| 取值范围             | \[1ms, 10s\] |
| 是否重启 OBServer 生效 | 否            |

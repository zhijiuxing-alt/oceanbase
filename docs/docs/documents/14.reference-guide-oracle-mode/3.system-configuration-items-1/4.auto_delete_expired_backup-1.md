auto_delete_expired_backup 
===============================================

auto_delete_expired_backup 用于设置是否开启自动删除过期备份的功能。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | * False   * True    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

election_cpu_quota 
=======================================

election_cpu_quota 用于设置给副本选举相关的后台工作分配的 CPU 配额。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值              | 3         |
| 取值范围             | \[0, 10\] |
| 是否重启 OBServer 生效 | 否         |

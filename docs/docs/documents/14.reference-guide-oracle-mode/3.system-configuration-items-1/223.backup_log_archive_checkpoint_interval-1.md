backup_log_archive_checkpoint_interval 
===========================================================

backup_log_archive_checkpoint_interval 用于设置备份数据备份 archivelog 的时间间隔。


|        属性        |     描述      |
|------------------|-------------|
| 参数类型             | 时间类型        |
| 默认值              | 0           |
| 取值范围             | \[0s, 24h\] |
| 是否重启 OBServer 生效 | 否           |

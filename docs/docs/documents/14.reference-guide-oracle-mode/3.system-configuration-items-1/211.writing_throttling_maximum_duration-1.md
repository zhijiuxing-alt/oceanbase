writing_throttling_maximum_duration 
========================================================

write_throttling_maximum_duration 通过控制内存分配进度，控制写入速度。即指定在触发写入限速后，剩余 memstore 内存分配完所需的时间。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 1h         |
| 取值范围             | \[1s, 3d\] |
| 是否重启 OBServer 生效 | 否          |

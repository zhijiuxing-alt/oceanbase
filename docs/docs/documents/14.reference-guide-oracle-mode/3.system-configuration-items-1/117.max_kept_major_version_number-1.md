max_kept_major_version_number 
==================================================

max_kept_major_version_number 用于设置数据保留的冻结版本数量。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 2         |
| 取值范围             | \[1, 16\] |
| 是否重启 OBServer 生效 | 否         |

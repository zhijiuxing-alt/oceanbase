user_block_cache_priority 
==============================================

user_block_cache_priority 用于设置数据块缓存在缓存系统中的优先级。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[1, +∞) |
| 是否重启 OBServer 生效 | 否        |

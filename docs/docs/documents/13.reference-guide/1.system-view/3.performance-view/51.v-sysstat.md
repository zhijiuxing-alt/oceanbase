v$sysstat 
==============================



功能 
-----------

租户级别的统计事件概况。

相关表/视图 
---------------

gv$sysstat

字段说明 
-------------



|  **字段名称**   |   **类型**    | **是否可以为 NULL** |   **描述**    |
|-------------|-------------|----------------|-------------|
| CON_ID      | bigint(20)  | NO             | 租户的 ID      |
| STATISTICS# | bigint(20)  | NO             | 统计事件的下标     |
| VALUE       | bigint(20)  | NO             | 统计事件对应的结果数值 |
| STAT_ID     | bigint(20)  | NO             | 统计事件的 ID    |
| NAME        | varchar(64) | NO             | 统计事件的名称     |
| CLASS       | bigint(20)  | NO             | 统计事件所属的类型   |

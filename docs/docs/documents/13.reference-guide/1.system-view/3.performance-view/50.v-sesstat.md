v$sesstat 
==============================



功能 
-----------

展示 Session 级别的统计事件概况。

相关表/视图 
---------------

gv$sesstat

字段说明 
-------------



|  **字段名称**   |   **类型**   | **是否可以为 NULL** |     **描述**     |
|-------------|------------|----------------|----------------|
| SID         | bigint(20) | NO             | 该 Session 的 ID |
| CON_ID      | bigint(20) | NO             | 租户的 ID         |
| STATISTICS# | bigint(20) | NO             | 统计事件的下标        |
| VALUE       | bigint(20) | NO             | 统计事件发生的次数      |

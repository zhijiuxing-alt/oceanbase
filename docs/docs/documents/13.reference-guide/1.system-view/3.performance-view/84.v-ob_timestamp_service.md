v$ob_timestamp_service 
===========================================



功能 
-----------

展示系统中各租户使用的时钟源和对应的值。

字段说明 
-------------



| **字段名称**  |   **类型**   | **是否可以为 NULL** |                                                                                     **描述**                                                                                      |
|-----------|------------|----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| tenant_id | bigint(20) | NO             | 租户 ID                                                                                                                                                                           |
| ts_type   | varchar(9) | NO             | 时钟源类型： * LOCAL   *  GLOBAL   * HA GLOBAL    |
| ts_value  | bigint(20) | NO             | 时钟值                                                                                                                                                                             |

mysql.time_zone_name 
=========================================





功能 
--------------------

用于记录 timezone 名称信息（SYSTEM_VIEW）。

相关表/视图 
------------------------

__all_time_zone_name

字段说明 
----------------------



|     字段名称     |     类型      | 是否可以为 NULL |  描述   |
|--------------|-------------|------------|-------|
| Name         | varchar(64) | NO         | 时区名称  |
| Time_zone_id | bigint(20)  | NO         | 时区 ID |

sql_safe_updates 
=====================================

sql_safe_updates 用于设置是否启用 MySQL 的 SQL 安全模式，默认为关闭。
**说明**



该变量仅用于兼容，OceanBase 数据库当前暂未实现该变量的功能。


| **属性**  |                                                          **描述**                                                          |
|---------|--------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                                     |
| 默认值     | 0                                                                                                                        |
| 取值范围    | * 0：表示 FALSE，即关闭   * 1：表示 TRUE，即开启    |
| 生效范围    | * GLOBAL   * SESSION                  |
| 是否参与序列化 | 是                                                                                                                        |

last_insert_id 
===================================

last_insert_id 用于返回本 Session 最后一次插入的自增字段值。


| **属性** |           **描述**            |
|--------|-----------------------------|
| 参数类型   | unit                        |
| 默认值    | 0                           |
| 取值范围   | \[0, 18446744073709551615\] |
| 生效范围   | SESSION                     |

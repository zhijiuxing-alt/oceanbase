datadir 
============================

datadir 用于设置数据存储的本地磁盘路径。
**说明**



该变量仅用于兼容，OceanBase 数据库当前暂未实现该变量的功能。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | varchar                                                                                                    |
| 默认值    | /usr/local/mysql/data/                                                                                     |
| 取值范围   | N/A                                                                                                        |
| 生效范围   | * GLOBAL   * SESSION    |
| 是否可修改  | 否                                                                                                          |

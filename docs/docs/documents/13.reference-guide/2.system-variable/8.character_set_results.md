character_set_results 
==========================================

character_set_results 用于设置服务器在将结果集或错误消息发送回客户端之前应转换为的字符集。


| **属性** |                                                                       **描述**                                                                        |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                                                                                                                                              |
| 默认值    | utf8mb4                                                                                                                                             |
| 取值范围   | * utf8mb4   * binary   * gbk    |
| 生效范围   | * GLOBAL   * SESSION                                             |
| 是否可以为空 | 是                                                                                                                                                   |

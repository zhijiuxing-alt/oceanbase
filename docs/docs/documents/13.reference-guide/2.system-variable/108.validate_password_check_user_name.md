validate_password_check_user_name 
======================================================

validate_password_check_user_name 用于设置用户密码是否可以和用户名相同。


| **属性**  |                                                              **描述**                                                               |
|---------|-----------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | enum                                                                                                                              |
| 默认值     | on                                                                                                                                |
| 取值范围    | * on：表示用户密码可以和用户名相同   * off：表示用户密码不可以和用户名相同    |
| 生效范围    | GLOBAL                                                                                                                            |
| 是否参与序列化 | 是                                                                                                                                 |

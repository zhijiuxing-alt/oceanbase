version_comment 
====================================

version_comment 用于设置 OBServer 的版本信息。


| **属性** |     **描述**      |
|--------|-----------------|
| 参数类型   | varchar         |
| 默认值    | OceanBase 1.0.0 |
| 取值范围   | N/A             |
| 生效范围   | GLOBAL          |
| 是否可修改  | 否               |

sql_mode 
=============================

sql_mode 用于设置 SQL 模式，不同的 SQL 模式对于插入等行为有很大影响。


| **属性** |                                                                                                                         **描述**                                                                                                                         |
|--------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | varchar                                                                                                                                                                                                                                                |
| 默认值    | STRICT_ALL_TABLES                                                                                                                                                                                                                                      |
| 取值范围   | * STRICT_ALL_TABLES   * STRICT_TRANS_TABLES   * ONLY_FULL_GROUP_BY   * PAD_CHAR_TO_FULL_LENGTH    |
| 生效范围   | * GLOBAL   * SESSION                                                                                                                                                |

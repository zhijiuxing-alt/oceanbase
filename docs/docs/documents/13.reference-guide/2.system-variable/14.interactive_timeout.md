interactive_timeout 
========================================

interactive_timeout 用于设置服务器关闭交互式连接前等待活动的秒数。交互式客户端定义为在 mysql_real_connect() 中使用 CLIENT_INTERACTIVE 选项的客户端。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                                                                                                        |
| 默认值    | 28800                                                                                                      |
| 取值范围   | \[1, 31536000\]                                                                                            |
| 生效范围   | * GLOBAL   * SESSION    |

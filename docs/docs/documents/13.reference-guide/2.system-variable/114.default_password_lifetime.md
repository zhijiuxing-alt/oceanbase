default_password_lifetime 
==============================================

default_password_lifetime 用于设置密码的过期时间。


| **属性**  |   **描述**    |
|---------|-------------|
| 参数类型    | uint        |
| 默认值     | 0           |
| 取值范围    | \[0,65535\] |
| 生效范围    | GLOBAL      |
| 是否参与序列化 | 是           |

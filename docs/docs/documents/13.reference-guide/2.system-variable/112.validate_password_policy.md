validate_password_policy 
=============================================

validate_password_policy 用于设置密码检查策略。


| **属性**  |                                                                                   **描述**                                                                                   |
|---------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | enum                                                                                                                                                                       |
| 默认值     | LOW                                                                                                                                                                        |
| 取值范围    | * LOW：表示仅包含密码常见的检测   * MEDIUM：表示包括密码长度检测、大写字母个数检测、小写字母个数检测、数字个数检测、特殊字符个数检测、用户名密码相同检测    |
| 生效范围    | GLOBAL                                                                                                                                                                     |
| 是否参与序列化 | 是                                                                                                                                                                          |

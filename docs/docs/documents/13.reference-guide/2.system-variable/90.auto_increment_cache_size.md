auto_increment_cache_size 
==============================================

auto_increment_cache_size 用于设置缓存的自增值个数。


| **属性**  |      **描述**      |
|---------|------------------|
| 参数类型    | int              |
| 默认值     | 1000000          |
| 取值范围    | \[1, 100000000\] |
| 生效范围    | GLOBAL           |
| 是否参与序列化 | 是                |

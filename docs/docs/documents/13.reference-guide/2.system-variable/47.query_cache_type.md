query_cache_type 
=====================================

query_cache_type 用于查询缓存类型。


| **属性** |                                                                                                              **描述**                                                                                                               |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | varchar                                                                                                                                                                                                                           |
| 默认值    | OFF                                                                                                                                                                                                                               |
| 取值范围   | * OFF：不缓存或检索结果。   * ON：缓存除了 `SELECT SQL_NO_CACHE ...`查询的所有结果。   * DEMAND：只缓存 `SELECT SQL_CACHE ... `查询的结果。    |
| 生效范围   | * GLOBAL   * SESSION                                                                                                                           |

ob_proxy_partition_hit 
===========================================

ob_proxy_partition_hit 用于设置 OBProxy 是否将 SQL 语句请求的数据发送到这些数据主 Partition 所在的 OBServer 上。


| **属性** |                                               **描述**                                                |
|--------|-----------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                |
| 默认值    | 1                                                                                                   |
| 取值范围   | * 1：是   * 0：否    |
| 生效范围   | SESSION                                                                                             |
| 是否可修改  | 否                                                                                                   |
| 是否可见   | 否，所有租户均不可见                                                                                          |

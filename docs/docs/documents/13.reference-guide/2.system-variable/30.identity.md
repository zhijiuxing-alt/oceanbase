identity 
=============================

identity 和变量 last_insert_id 是同义词，可以通过 `select @@identity` 查询。


| **属性** |           **描述**            |
|--------|-----------------------------|
| 参数类型   | uint                        |
| 默认值    | 0                           |
| 取值范围   | \[0, 18446744073709551615\] |
| 生效范围   | SESSION                     |

ob_proxy_set_trx_executed 
==============================================

ob_proxy_set_trx_executed 用于设置用户是否通过 OBProxy 发送 `START TRANSACTION Syntax` 给 OBServer。


| **属性** |                                                 **描述**                                                 |
|--------|--------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                   |
| 默认值    | 0                                                                                                      |
| 取值范围   | * 0：不发送   * 1：发送    |
| 生效范围   | SESSION                                                                                                |
| 是否可修改  | 否                                                                                                      |
| 是否可见   | 否，所有租户均不可见                                                                                             |

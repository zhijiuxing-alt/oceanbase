parallel_max_servers 
=========================================

parallel_max_servers 用于设置每个 Server 上并行执行（Parallel eXecution，PX）线程池的大小。


| **属性**  |   **描述**    |
|---------|-------------|
| 参数类型    | int         |
| 默认值     | 0           |
| 取值范围    | \[0, 1800\] |
| 生效范围    | GLOBAL      |
| 是否参与序列化 | 是           |

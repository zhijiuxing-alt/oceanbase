validate_password_special_char_count 
=========================================================

validate_password_special_char_count 用于设置用户密码至少包含的特殊字符个数。


| **属性**  |      **描述**      |
|---------|------------------|
| 参数类型    | uint             |
| 默认值     | 0                |
| 取值范围    | \[0,2147483647\] |
| 生效范围    | GLOBAL           |
| 是否参与序列化 | 是                |

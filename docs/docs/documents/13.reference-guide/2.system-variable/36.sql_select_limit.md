sql_select_limit 
=====================================

sql_select_limit 用于设置一个 select 查询允许返回的最大行数。如果一个 select 中包含了 limit 语句，那么 limit 的值比 sql_select_limit 的优先级高，默认不限制 select 返回的行数。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | 9223372036854775807                                                                                        |
| 取值范围    | \[0, 9223372036854775807\]                                                                                 |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |

max_allowed_packet 
=======================================

max_allowed_packet 用于设置最大网络包大小，单位是 Byte。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | 4194304                                                                                                    |
| 取值范围    | \[1024,1073741824\]                                                                                        |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |




**说明**



该变量不能通过 `ALTER SESSION SET xxx = yyy; `语句的方式使其仅在当前 Session 生效，只能通过 `SET GLOBAL xxx = yyy` 语句这种全局生效的修改方式。 使用时一般客户端与 Server 端均需要调整。

ob_query_timeout 
=====================================

ob_query_timeout 用于设置查询超时时间，单位是微秒。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | 10000000                                                                                                   |
| 取值范围    | NULL                                                                                                       |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |

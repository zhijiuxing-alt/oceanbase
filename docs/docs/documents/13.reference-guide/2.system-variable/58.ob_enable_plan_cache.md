ob_enable_plan_cache 
=========================================

ob_enable_plan_cache 用于设置是否打开 Plan Cache。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 1                                                                                                          |
| 取值范围    | * 0：关闭   * 1：打开         |
| 生效范围    | * GLOBAL   * SESSION    |
| 是否参与序列化 | 是                                                                                                          |

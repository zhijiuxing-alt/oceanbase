character_set_server 
=========================================

character_set_server 用于设置服务器字符集。


| **属性** |                                                                                                                   **描述**                                                                                                                    |
|--------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                                                                                                                                                                                                                                      |
| 默认值    | utf8mb4                                                                                                                                                                                                                                     |
| 取值范围   | * utf8mb4   * binary   * gbk   * gb18030   * utf16    |
| 生效范围   | * GLOBAL   * SESSION                                                                                                                                     |
| 是否可以为空 | 是                                                                                                                                                                                                                                           |

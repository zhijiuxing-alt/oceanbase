sql_warnings 
=================================

sql_warnings 用于在出现告警时控制单行 INSERT 语句是否生成信息字符串。


| **属性** |                                                        **描述**                                                         |
|--------|-----------------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                                  |
| 默认值    | OFF                                                                                                                   |
| 取值范围   | * ON：生成一个信息字符串   * OFF：不生成信息字符串    |
| 生效范围   | * GLOBAL   * SESSION               |

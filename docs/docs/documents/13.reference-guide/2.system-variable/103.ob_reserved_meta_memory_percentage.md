ob_reserved_meta_memory_percentage 
=======================================================

ob_reserved_meta_memory_percentage 用于设置租户预留内存的比例，主要存储 Meta 相关的结构信息。


| **属性**  |  **描述**  |
|---------|----------|
| 参数类型    | int      |
| 默认值     | 10       |
| 取值范围    | \[1,100） |
| 生效范围    | GLOBAL   |
| 是否参与序列化 | 是        |

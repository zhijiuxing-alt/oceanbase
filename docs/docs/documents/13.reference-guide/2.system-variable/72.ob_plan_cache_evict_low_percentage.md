ob_plan_cache_evict_low_percentage 
=======================================================

ob_plan_cache_evict_low_percentage 用于设置停止计划缓存逐出的计划缓存限制的内存使用百分比。


| **属性**  |   **描述**   |
|---------|------------|
| 参数类型    | int        |
| 默认值     | 50         |
| 取值范围    | \[0, 100\] |
| 生效范围    | GLOBAL     |
| 是否参与序列化 | 是          |

dtl_buffer_size 
====================================

dtl_buffer_size 用于设置 SQL 数据传输模块使用的缓存大小。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 容量单位        |
| 默认值              | 64K         |
| 取值范围             | \[4K, 2 M\] |
| 是否重启 OBServer 生效 | 否           |

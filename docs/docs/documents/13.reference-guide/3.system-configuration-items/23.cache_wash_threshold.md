cache_wash_threshold 
=========================================

cache_wash_threshold 用于设置触发缓存清理的容量阈值。如果内存空间小于指定值时，内存空间将被清理。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 容量单位     |
| 默认值              | 4GB      |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 生效 | 否        |

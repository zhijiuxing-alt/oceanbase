merger_check_interval 
==========================================

merger_check_interval 用于设置合并状态检查线程的调度间隔。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 10m，表示 10 分钟 |
| 取值范围             | \[10s, 60m\] |
| 是否重启 OBServer 生效 | 否            |

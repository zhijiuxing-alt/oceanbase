rootservice_memory_limit 
=============================================

rootservice_memory_limit 用于设置 RootService 的最大内存容量限制。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 2G        |
| 取值范围             | \[2G, +∞) |
| 是否重启 OBServer 生效 | 否         |

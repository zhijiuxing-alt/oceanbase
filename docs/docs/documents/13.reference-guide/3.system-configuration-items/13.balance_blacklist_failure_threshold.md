balance_blacklist_failure_threshold 
========================================================

balance_blacklist_failure_threshold 用于设置副本迁移等后台任务连续失败次数的阈值。副本迁移等后台任务连续失败次数超过该阈值后，将被放入黑名单。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 5           |
| 取值范围             | \[0, 1000\] |
| 是否重启 OBServer 生效 | 否           |

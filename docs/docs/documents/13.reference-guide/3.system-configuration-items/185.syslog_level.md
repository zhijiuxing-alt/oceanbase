syslog_level 
=================================

syslog_level 用于设置系统日志级别。


|      **属性**      |                                                                                                                                         **描述**                                                                                                                                         |
|------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                                                                                                                                                                    |
| 默认值              | INFO                                                                                                                                                                                                                                                                                   |
| 取值范围             | * DEBUG   * TRACE   * INFO   * WARN   * USER_ERR   * ERROR\]    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                                                                      |

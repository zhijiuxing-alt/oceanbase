clog_transport_compress_func 
=================================================

`clog_transport_compress_func` 用于设置事务日志内部传输时使用的压缩算法。


|      **属性**      |                                                                                                                                                                                                                                **描述**                                                                                                                                                                                                                                |
|------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 默认值              | lz4_1.0                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 取值范围             | * none   * lz4_1.0   * snappy_1.0   * zlib_1.0   * zstd_1.0   * stream_lz4_1.0   * stream_zstd_1.0   * zstd_1.3.8   * stream_zstd_1.3.8    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |

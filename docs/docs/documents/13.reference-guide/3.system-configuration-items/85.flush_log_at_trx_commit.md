flush_log_at_trx_commit 
============================================

flush_log_at_trx_commit 用于设置事务提交时写事务日志策略。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[0, 2\] |
| 是否重启 OBServer 生效 | 否        |



flush_log_at_trx_commit 配置项的取值含义如下：

* 取值为 0 表示不等待日志写入缓冲区。

  

* 取值为 1 表示等待日志写入磁盘。

  

* 取值为 2 表示等待日志写入缓冲区而不等落盘。

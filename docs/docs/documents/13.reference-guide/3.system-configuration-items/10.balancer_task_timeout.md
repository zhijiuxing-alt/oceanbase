balancer_task_timeout 
==========================================

balancer_task_timeout 用于设置负载均衡等后台任务的超时时间。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 20m，表示 20 分钟 |
| 取值范围             | \[1s, +∞)    |
| 是否重启 OBServer 生效 | 否            |

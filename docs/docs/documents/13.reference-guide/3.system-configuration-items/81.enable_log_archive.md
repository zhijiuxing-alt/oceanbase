enable_log_archive 
=======================================

enable_log_archive 用于设置是否开启日志归档功能。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | * True   * False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

global_index_build_single_replica_timeout 
==============================================================

global_index_build_single_replica_timeout 用于建全局索引时，设置每个副本构建的超时时间。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 48h       |
| 取值范围             | \[1h, +∞) |
| 是否重启 OBServer 生效 | 否         |

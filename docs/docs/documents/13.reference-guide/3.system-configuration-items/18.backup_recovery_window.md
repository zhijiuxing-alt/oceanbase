backup_recovery_window 
===========================================

backup_recovery_window 用于设置备份数据可以恢复的时间。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 0        |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 生效 | 否        |


**说明**



备份数据恢复的时间单位为天，默认值为 0，表示不自动清理。

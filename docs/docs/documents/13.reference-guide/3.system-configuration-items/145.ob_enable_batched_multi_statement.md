ob_enable_batched_multi_statement 
======================================================

ob_enable_batched_multi_statement 用于设置是否启用批处理多条语句的功能。


|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                          |
| 默认值              | False                                                                                                         |
| 取值范围             | * False：不启用   * True：启用    |
| 是否重启 OBServer 生效 | 否                                                                                                             |

enable_one_phase_commit 
============================================

enable_one_phase_commit 用于设置是否开启事务提交一阶段优化的功能。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | * True   * False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |

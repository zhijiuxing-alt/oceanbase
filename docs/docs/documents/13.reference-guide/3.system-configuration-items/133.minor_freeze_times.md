minor_freeze_times 
=======================================

minor_freeze_times 用于设置多少次小合并触发一次全局合并。值为 0 时，表示关闭小合并。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 5            |
| 取值范围             | \[0, 65535\] |
| 是否重启 OBServer 生效 | 否            |


**说明**



minor_freeze_times 配置项与 major_compact_trigger 配置项具有相同功能。

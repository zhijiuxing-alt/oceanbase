server_balance_disk_tolerance_percent 
==========================================================

server_balance_disk_tolerance_percent 用于设置节点间磁盘负载均衡策略中不均衡程度的容忍度。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 1          |
| 取值范围             | \[1, 100\] |
| 是否重启 OBServer 生效 | 否          |

enable_record_trace_log 
============================================

enable_record_trace_log 用于设置是否记录追踪日志。


|      **属性**      |                                                                                        **描述**                                                                                        |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                                                                                                 |
| 默认值              | False                                                                                                                                                                                |
| 取值范围             | * True：表示只要出现错误，就一定打印 trace log。   * False：表示每隔 trace_log_sampling_interval 会打印一次慢查询或白名单错误码日志。    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                    |

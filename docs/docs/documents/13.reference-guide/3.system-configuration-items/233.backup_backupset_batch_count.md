backup_backupset_batch_count 
=================================================

backup_backupset_batch_count 用于设置备份备份基线的任务的批次数量。


|        属性        |     描述     |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 0          |
| 取值范围             | \[0,1024\] |
| 是否重启 OBServer 生效 | 否          |

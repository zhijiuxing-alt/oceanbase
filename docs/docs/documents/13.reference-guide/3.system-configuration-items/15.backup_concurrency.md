backup_concurrency 
=======================================

backup_concurrency 用于设置备份时写文件系统的并发数量。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 0          |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 生效 | 否          |

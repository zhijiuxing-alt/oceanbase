default_compress_func 
==========================================

`default_compress_func` 用于设置表数据的默认压缩算法，也可以在建表的时候指定其他压缩算法。


|      **属性**      |                                                                                                                                                **描述**                                                                                                                                                |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                                                                                                                                                                                  |
| 默认值              | zstd_1.3.8                                                                                                                                                                                                                                                                                           |
| 取值范围             | * none   * lz4_1.0   * snappy_1.0   * zlib_1.0   * zstd_1.0   * zstd_1.3.8    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                                                                                    |

clog_disk_usage_limit_percentage 
=====================================================

clog_disk_usage_limit_percentage 用于设置事务日志的磁盘 I/O 最大使用百分比。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 95          |
| 取值范围             | \[80, 100\] |
| 是否重启 OBServer 生效 | 否           |

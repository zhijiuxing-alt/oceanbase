rootservice_async_task_queue_size 
======================================================

rootservice_async_task_queue_size 用于设置 RootService 内部异步任务队列的大小。


|      **属性**      |    **描述**     |
|------------------|---------------|
| 参数类型             | 整型            |
| 默认值              | 16384         |
| 取值范围             | \[8, 131072\] |
| 是否重启 OBServer 生效 | 否             |

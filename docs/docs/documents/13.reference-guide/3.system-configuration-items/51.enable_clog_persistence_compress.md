enable_clog_persistence_compress 
=====================================================

enable_clog_persistence_compress 用于设置是否开启事务日志落盘压缩。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | false  |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |

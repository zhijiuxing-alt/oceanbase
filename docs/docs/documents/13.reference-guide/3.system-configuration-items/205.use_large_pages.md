use_large_pages 
====================================

use_large_pages 用于管理数据库使用的内存大页。


|      **属性**      |                                                                      **描述**                                                                      |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                              |
| 默认值              | false                                                                                                                                            |
| 取值范围             | * false   * true   * only    |
| 是否重启 OBServer 生效 | 是                                                                                                                                                |

config_additional_dir 
==========================================

config_additional_dir 用于设置本地存储配置文件的多个目录，为了冗余存储多份配置文件。


|      **属性**      |                                                **描述**                                                 |
|------------------|-------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串列表                                                                                                 |
| 默认值              | * etc2   * etc3    |
| 取值范围             | NULL                                                                                                  |
| 是否重启 OBServer 生效 | 否                                                                                                     |

sys_bkgd_io_timeout 
========================================

sys_bkgd_io_timeout 系统后台 IO 请求超时时间。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 120s         |
| 取值范围             | \[5s, 600s\] |
| 是否重启 OBServer 生效 | 否            |

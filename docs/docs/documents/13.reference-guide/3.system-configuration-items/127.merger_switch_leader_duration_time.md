merger_switch_leader_duration_time 
=======================================================

merger_switch_leader_duration_time 用于设置每日合并时批量切主的时间间隔。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 3m，表示 3 分钟 |
| 取值范围             | \[0s, 30\] |
| 是否重启 OBServer 生效 | 否          |

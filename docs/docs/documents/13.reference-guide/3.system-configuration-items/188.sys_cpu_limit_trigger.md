sys_cpu_limit_trigger 
==========================================

sys_cpu_limit_trigger 用于设置 CPU 利用率阈值，即当 CPU 利用率超过该阈值时，将暂停系统后台任务的执行。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 80        |
| 取值范围             | \[50, +∞) |
| 是否重启 OBServer 生效 | 否         |

tableapi_transport_compress_func 
=====================================================

`tableapi_transport_compress_func` 用于设置 tableAPI 查询结果传输使用的压缩算法。


|      **属性**      |                                                                                                                                                **描述**                                                                                                                                                |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                                                                                                                |
| 默认值              | none                                                                                                                                                                                                                                                                                                 |
| 取值范围             | * none   * lz4_1.0   * snappy_1.0   * zlib_1.0   * zstd_1.0   * zstd 1.3.8    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                                                                                    |

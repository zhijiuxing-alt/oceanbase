clog_persistence_compress_func 
===================================================

clog_persistence_compress_func 用于设置事务日志落盘压缩算法。


|      **属性**      |                                                                                                 **描述**                                                                                                 |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                                                                                    |
| 默认值              | lz4_1.0                                                                                                                                                                                                |
| 取值范围             | * none   * lz4_1.0   * zstd_1.0   * zstd_1.3.8    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                      |

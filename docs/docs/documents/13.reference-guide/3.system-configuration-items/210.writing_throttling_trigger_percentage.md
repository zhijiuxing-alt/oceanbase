writing_throttling_trigger_percentage 
==========================================================

write_throttling_trigger_percentage 用于设置写入速度的阈值。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 100        |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 生效 | 否          |



当 memstore 已使用的内存达到该阈值时，触发写入限速。当该配置项的值为 100 时，表示关闭写入限速机制。

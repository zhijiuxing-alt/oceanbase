mysql_port 
===============================

mysql_port 用于设置 SQL 服务协议端口号。


|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | 整型              |
| 默认值              | 2880            |
| 取值范围             | \[1025, 65535\] |
| 是否重启 OBServer 生效 | 否               |

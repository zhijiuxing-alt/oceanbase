clog_max_unconfirmed_log_count 
===================================================

clog_max_unconfirmed_log_count 用于设置事务模块中未确认日志的最大数量。


|      **属性**      |     **描述**     |
|------------------|----------------|
| 参数类型             | 整型             |
| 默认值              | 1500           |
| 取值范围             | \[100, 50000\] |
| 是否重启 OBServer 生效 | 否              |

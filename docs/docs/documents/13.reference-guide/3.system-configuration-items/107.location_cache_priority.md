location_cache_priority 
============================================

location_cache_priority 用于设置位置缓存在系统缓存服务中的优先级。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 1000      |
| 取值范围             | \[1s, +∞) |
| 是否重启 OBServer 生效 | 否         |

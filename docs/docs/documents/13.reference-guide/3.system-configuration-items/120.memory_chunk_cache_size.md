memory_chunk_cache_size 
============================================

memory_chunk_cache_size 用于设置内存分配器缓存的内存块容量。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 0M        |
| 取值范围             | \[0M, +∞) |
| 是否重启 OBServer 生效 | 否         |



当 memory_chunk_cache_size 配置项的值为 0 时，表示系统自适应。

clog_disk_utilization_threshold 
====================================================

clog_disk_utilization_threshold 配置项用于控制 clog 或 ilog 磁盘空间复用的水位值。

描述 
-----------------------



|        属性        |     描述     |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 80         |
| 取值范围             | \[10, 100) |
| 是否重启 OBServer 生效 | 否          |



示例 
-----------------------

```sql
obclient> ALTER SYSTEM SET clog_disk_utilization_threshold=80;
```

px_workers_per_cpu_quota 
=============================================

px_workers_per_cpu_quota 用于设置并行执行工作线程数的比例。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 10        |
| 取值范围             | \[0, 20\] |
| 是否重启 OBServer 生效 | 否         |

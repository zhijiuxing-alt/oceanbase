location_cache_refresh_min_interval 
========================================================

location_cache_refresh_min_interval 用于设置位置缓存刷新请求的最小间隔，防止产生过多刷新请求造成系统压力过大。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 100ms     |
| 取值范围             | \[0s, +∞) |
| 是否重启 OBServer 生效 | 否         |

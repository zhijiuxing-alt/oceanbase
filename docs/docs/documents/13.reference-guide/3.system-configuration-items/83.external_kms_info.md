external_kms_info 
======================================

external_kms_info 用于存储一些密钥管理信息。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |



使用外部密钥管理中心时，external_kms_info 将存储一些密钥管理信息。

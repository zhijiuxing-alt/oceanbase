tde_method 
===============================

tde_method 用于设置透明表空间加密的方式。


|      **属性**      |                                                                                                **描述**                                                                                                |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                |
| 默认值              | none                                                                                                                                                                                                 |
| 取值范围             | * none：表示无法使用透明表空间加密   * internal：表示透明加密采用内部表的形式   * bkmi：表示透明加密采用外部 bkmi 的形式    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                    |

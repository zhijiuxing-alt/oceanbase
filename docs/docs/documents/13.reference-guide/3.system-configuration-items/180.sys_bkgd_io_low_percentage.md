sys_bkgd_io_low_percentage 
===============================================

sys_bkgd_io_low_percentage 用于设置系统后台 IO 最少占用的百分比。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 0          |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 生效 | 否          |



当 sys_bkgd_io_low_percentage 的值为 0 时，系统根据环境自动配置比例。

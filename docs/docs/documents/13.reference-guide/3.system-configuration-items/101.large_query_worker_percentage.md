large_query_worker_percentage 
==================================================

large_query_worker_percentage 用于设置预留给处理大查询的工作线程百分比。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 双精度浮点数     |
| 默认值              | 30         |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 生效 | 否          |

minor_merge_concurrency 
============================================

minor_merge_concurrency 用于设置小合并时的并发线程数。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 0         |
| 取值范围             | \[0, 64\] |
| 是否重启 OBServer 生效 | 否         |

disk_io_thread_count 
=========================================

disk_io_thread_count 用于设置磁盘 IO 线程数。必须为偶数。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 8         |
| 取值范围             | \[2, 32\] |
| 是否重启 OBServer 生效 | 否         |

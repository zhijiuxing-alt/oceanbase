switchover_process_thread_count 
====================================================

switchover_process_thread_count 用于设置主备库切换相关线程池的大小。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 6           |
| 取值范围             | \[1, 1000\] |
| 是否重启 OBServer 生效 | 否           |

通过 obclient 探索 OceanBase MySQL 租户 
======================================================



在 obclient 命令行环境里，可以通过一些命令或者 SQL 查看数据库对象或者表属性和数据。

示例 
-----------

* 通过 obclient 查看 MySQL 租户的数据库对象。




```javascript
#obclient -h192.168.1.101 -utpcc@obmysql#obdemo -P2883 -p123456 -A tpccdb

obclient> show tables;

```

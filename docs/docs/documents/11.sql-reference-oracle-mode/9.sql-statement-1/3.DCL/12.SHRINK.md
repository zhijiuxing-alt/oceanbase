SHRINK 
===========================



描述 
-----------

该语句会将表的数据重整，即在合并的过程中重新写一遍。

格式 
-----------

```javascript
ALTER TABLE table_name SHRINK SPACE
```



参数解释 
-------------



|     参数     | 描述 |
|------------|----|
| table_name | 表名 |



示例 
-----------

* 将表 t1 的数据重整

  




```javascript
OceanBase(admin@test)>ALTER TABLE t1 SHRINK SPACE;
```

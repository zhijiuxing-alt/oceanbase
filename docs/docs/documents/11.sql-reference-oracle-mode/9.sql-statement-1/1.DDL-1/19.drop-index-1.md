DROP INDEX 
===============================



描述 
-----------

该语句用来删除索引。当索引过多时，维护开销增大，因此，需要删除不必要的索引。

删除索引的时候需要等待一段时间才能完全删除。

格式 
-----------

```javascript
DROP INDEX [schema.]indexname;
```



参数解释 
-------------



|    参数     |      描述      |
|-----------|--------------|
| schema    | 指定 schema 名称 |
| indexname | 指定索引名称       |



示例 
-----------

* 删除索引 test_index。

  




```javascript
DROP INDEX test_index;
```

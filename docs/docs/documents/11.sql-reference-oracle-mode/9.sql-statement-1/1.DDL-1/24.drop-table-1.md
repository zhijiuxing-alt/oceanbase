DROP TABLE 
===============================



描述 
-----------

该语句用于删除 OceanBase 数据库中的表。

格式 
-----------

```javascript
DROP TABLE table_name [CASCADE CONSTRAINTS] [PURGE]
```



参数解释 
-------------



|         参数          |           描述           |
|---------------------|------------------------|
| table_name          | 指定要删除的表名               |
| CASCADE CONSTRAINTS | 级联删除跟 table_name 关联的约束 |
| PURGE               | 彻底删除表 (不进回收站)          |



示例 
-----------

* 删除表 test

  




```javascript
DROP TABLE test;
```

RENAME 
===========================



描述 
-----------

该语句用来对对象进行重命名。

格式 
-----------

```javascript
RENAME obj_name TO new_obj_name;
```



参数解释 
-------------



|      参数      |  描述   |
|--------------|-------|
| obj_name     | 原对象名。 |
| new_obj_name | 新对象名。 |



示例 
-----------

1. 创建表 t1 和 t2。

   




```javascript
create table t1(c1 int);
```



2. 将表 t1 重命名为 t11。

   




```javascript
rename t1 to t11;
```



注意事项 
-------------

重命名操作可以对表、视图、私有同义词、sequence 进行。

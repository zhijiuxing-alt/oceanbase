DROP OUTLINE 
=================================



描述 
-----------

该语句用于删除 OceanBase 数据库中的 OUTLINE。

格式 
-----------

```javascript
DROP OUTLINE outline_name；  
```



参数解释 
-------------



|      参数      |         描述         |
|--------------|--------------------|
| outline_name | 指定要删除的 OUTLINE 名称。 |



示例 
-----------

* 删除 OUTLINE ol_1

  




```javascript
DROP OUTLINE ol_1;      
```

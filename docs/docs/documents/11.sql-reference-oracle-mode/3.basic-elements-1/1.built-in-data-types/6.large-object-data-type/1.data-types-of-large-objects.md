大对象数据类型概述 
==============================



大对象数据类型（Large Object），简称 LOB，用来存储大型和非结构化数据，例如文本、图像、视频和空间数据等。包括 `BLOB` 和 `CLOB` 。

您在创建表时，可以选择为 LOB 列或 LOB 对象属性指定与表中指定的不同的表空间和存储特征。

OceanBase 支持以下大对象数据类型：


|  类型  | 长度 | **定义长度上限（字符）** |   字符集   |
|------|----|----------------|---------|
| BLOB | 变长 | 48M            | BINARY  |
| CLOB | 变长 | 48M            | UTF8MB4 |

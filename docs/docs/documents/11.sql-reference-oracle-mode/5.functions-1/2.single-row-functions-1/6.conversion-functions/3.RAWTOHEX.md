RAWTOHEX 
=============================



`RAWTOHEX` 函数将二进制数转换为一个相应的十六进制表示的字符串。

语法 
--------------

```javascript
RAWTOHEX (raw)
```



参数 
--------------



| 参数  |    说明    |
|-----|----------|
| raw | 二进制的字符串。 |



返回类型 
----------------

十六进制表示的字符串。

示例 
--------------

执行以下语句：

```javascript
SELECT RAWTOHEX('AB')  FROM DUAL;
```



查询结果如下：

```javascript
+----------------+
| RAWTOHEX('AB') |
+----------------+
| 4142           |
+----------------+
```

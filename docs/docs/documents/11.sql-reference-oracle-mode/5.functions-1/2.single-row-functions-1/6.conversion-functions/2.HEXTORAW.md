HEXTORAW 
=============================



`HEXTORAW` 函数将 `CHAR`、`VARCHAR2`、`NCHAR` 或 `NVARCHAR2` 数据类型中包含十六进制数字的字符转换为 `RAW` 数据类型。

语法 
--------------

```javascript
HEXTORAW (char)
```



参数 
--------------



|  参数  |                            说明                             |
|------|-----------------------------------------------------------|
| char | 十六进制的字符串。字符串类型可为：`CHAR`、`VARCHAR2`、`NCHAR` 或 `NVARCHAR2`。 |



返回类型 
----------------

返回 `RAW` 类型的数据。

示例 
--------------

执行以下语句：

```javascript
select HEXTORAW('A123')  from dual;
```



查询结果如下：

```javascript
+------------------+
| HEXTORAW('A123') |
+------------------+
| A123             |
+------------------+
```

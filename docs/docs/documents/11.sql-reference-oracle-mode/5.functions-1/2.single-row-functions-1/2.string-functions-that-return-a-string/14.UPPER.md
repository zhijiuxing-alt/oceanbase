UPPER 
==========================



`UPPER` 函数将字符串全部转为大写。

语法 
--------------

```javascript
UPPER(c1)
```



参数 
--------------



| 参数 |                             说明                              |
|----|-------------------------------------------------------------|
| c1 | 字符串。字符串类型可为：`CHAR`、`VARCHAR2`、`NCHAR`、`NVARCHAR2` 或 `CLOB`。 |



返回类型 
----------------

返回 `CHAR`、`VARCHAR2`、`NCHAR`、`NVARCHAR2` 或 `CLOB` 类型的字符。

示例 
--------------

执行以下语句：

```javascript
SELECT UPPER('AaBbCcDd') upper FROM DUAL;
```



查询结果如下：

```javascript
+----------+
| UPPER    |
+----------+
| AABBCCDD |
+----------+
```

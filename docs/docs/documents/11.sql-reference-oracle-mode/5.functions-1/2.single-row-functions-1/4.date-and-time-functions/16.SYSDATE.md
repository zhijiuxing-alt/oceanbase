SYSDATE 
============================



`SYSDATE` 函数返回当前日期。
**注意**



该函数不依赖于当前会话时区，而是依赖于数据库宿主机的 Linux 操作系统的时区。

语法 
--------------

```javascript
SYSDATE
```



参数 
--------------

无参数。

返回类型 
----------------

`DATE` 数据类型。

示例 
--------------

以下示例将当前时间按指定格式输出：

```javascript
SELECT TO_CHAR
(SYSDATE, 'MM-DD-YYYY HH24:MI:SS') "NOW"
FROM DUAL;
```



查询结果如下：

```javascript
+---------------------+
| NOW                 |
+---------------------+
| 03-08-2020 20:44:04 |
+---------------------+
```

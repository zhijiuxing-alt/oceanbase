Oracle 模式支持的数据类型 
=====================================

Oracle 模式所支持的数据类型如下列表所示。

* 字符型

  * VARCHAR2

    
  
  * NVARCHAR2

    
  
  * CHAR

    
  
  * NCHAR

    
  

  




<!-- -->

* 数字类型支持 

  * NUMBER

    
  
  * NUMBER_FLOAT

    
  
  * BINARY_FLOAT

    
  
  * BINARY_DOUBLE

    
  

  




<!-- -->

* 复杂数据类型

  * Ref Cursor

    
  
  * Struct

    
  
  * Array

    
  
  * RAW

    
  
  * RowId

    
  

  




<!-- -->

* 日期型

  * DATE

    
  
  * TIMESTAMP

    
  
  * TIMESTAMP WITH TIME ZONE

    
  
  * TIMESTAMP WITH LOCAL TIME ZONE

    
  
  * INTERVAL

    
  
  * INTERVALDS

    
  
  * INTERVALYM

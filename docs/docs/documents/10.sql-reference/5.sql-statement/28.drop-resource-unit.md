DROP RESOURCE UNIT 
=======================================



描述 
-----------

该语句用来删除资源单元配置。

格式 
-----------

```javascript
DROP RESOURCE UNIT unitname 
```



参数解释 
-------------



|  **参数**  |    **描述**     |
|----------|---------------|
| unitname | 要删除的资源单元配置的名称 |





示例 
-----------

* 删除资源单元配置unit1。




```javascript
OceanBase(admin@test)> DROP RESOURCE UNIT unit1;
Query OK, 0 rows affected (0.00 sec)
```

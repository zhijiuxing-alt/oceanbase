DROP RESOURCE POOL 
=======================================



描述 
-----------

该语句用来删除资源池。

格式 
-----------

```javascript
DROP RESOURCE POOL poolname; 
```



参数解释 
-------------



|  **参数**  |  **描述**   |
|----------|-----------|
| poolname | 要删除的资源池名称 |



示例 
-----------

* 删除资源池 pool1。




```javascript
OceanBase(admin@test)> DROP RESOURCE POOL pool1;
Query OK, 0 rows affected (0.00 sec)
```

日期时间运算符 
============================



以下为所有日期时间运算符列表：


| 运算符 | 操作数 |      含义       |
|-----|-----|---------------|
| `+` | 二元  | 同`DATE_ADD`函数 |
| `-` | 二元  | 同`DATE_SUB`函数 |



例如：

```javascript
OceanBase (root@oceanbase)> SELECT '2008-12-31 23:59:59' + INTERVAL 1 SECOND;
+-------------------------------------------+
| '2008-12-31 23:59:59' + INTERVAL 1 SECOND |
+-------------------------------------------+
| 2009-01-01 00:00:00                       |
+-------------------------------------------+
1 row in set (0.01 sec)
```

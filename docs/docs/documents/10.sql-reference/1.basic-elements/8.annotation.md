注释 
=======================



SQL 语句 
---------------

在普通 SQL 语句中，OceanBase 数据库支持如下 3 种注释方法：

* 从 `#` 到行尾

  




<!-- -->

* 从 `--` 到行尾

  

* 从 `/*` 到 `*/`

  






数据库对象 
--------------

在 DDL 语句中通过 `COMMENT` 子句可以为数据库对象指定注释。例如：

```unknow
create table t(pk INT PRIMARY KEY COMMENT '主键');
```
